# Crime Prevention Portal

This project is a Crime Prevention Portal website designed to spread awareness about crime safety, provide preventive measures, and guide users on how to stay safe in different situations. The website aims to deliver important safety information through a clean, informative, and user-friendly interface.

# Technologies Used
- HTML
- Tailwind CSS

# Features
- Informative crime prevention guidelines
- Safety tips and awareness sections
- User-friendly and responsive design
- Organized crime category information
- Clean and modern UI layout
- Easy navigation between safety resources

# Sections Included
- Home
- Crime Awareness
- Prevention Tips
- Safety Guidelines
- Emergency Information
- Contact / Help Section

# How to Run
1. Clone the repository:
   git clone https://github.com/lakshminarayanverma91-bot/CrimePreventionPortal

2. Open the project folder:
   cd CrimePreventionPortal

3. Open index.html in your browser

- To provide safety knowledge and guidelines
